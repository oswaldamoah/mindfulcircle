# Mindful Circle

A modern community platform for <a href="https://mindfulcircle.vercel.app">**Mindful Circle**</a>, making it easy for people to discover events, connect with the community, and support the circle.

## Features
The platform allows users to:

- View upcoming and past events
- Register for events online
- Explore photo galleries from community activities
- Support the Mindful Circle through volunteering and donations
